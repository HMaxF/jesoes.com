Jesoes.com provides a convenient way to explore and study the Christian Bible. Here's what makes it unique:

  1. Friendly navigation: Easily follow along with Bible reading, discussion or studies that reference multiple verses.
  2. Automatically save all read verses: Easy to jump back-and-forth between multiple books and chapters through bookmark.
  3. Read multiple Bible translations simultaneously: Compare different Bible translations to gain a deeper understanding of scripture.
  4. Designed for small screen: The web design is suitable for any screen size of smartphones, tablets or desktop.
  5. Cache locally: All Bible content are downloaded and stored in-device for fast reading.
  6. Open Bible Format (OBF): An open data format that is a text in JSON format, it can store the whole Bible, easy to create, read and parse by machine, anyone can create and share Bible translations.

How to use Jesoes.com.

  • "Choose" tab
    • Choose the book name
    • Choose the chapter number
    • Choose the verse number
    • See the preview of the verse text
    • Click 'Go' button to see verse
  • "Read" tab
    • Scroll through to read verses.
    • To compare Bible translations click the verse to open the other Bible translations (eg: English, Indonesian, Korean, etc.).
    • Select the other Bible translations from the left menu.
  • "Previous" tab
    • All read verses will be automatically stored and displayed here.
    • See when (date and time) and what verses that you previously read.
    • Click the item to read it again.

Additional Notes:

  • Jesoes.com does not collect any data at all.
  • This website operation is fast because the OBF data is cached and stored in this device only.
  • Please consider to contribute to create new Bible translations through the OBF format.
